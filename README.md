# Appscrip-task-Sumit_Shakya

A responsive Product Listing Page (PLP) built to strictly match provided Figma designs under tight performance constraints.

### Tech Stack
- Next.js 14 (App Router)
- React.js
- Vanilla CSS / CSS Modules
- Fake Store API (Server-Side Rendering)

### Installation & Setup

1. **Clone the repository:**
```bash
git clone <repository_url>
cd Appscrip-task-Sumit_Shakya
```

2. **Install dependencies:**
```bash
npm install
```

3. **Start the development server:**
```bash
npm run dev
```

4. **View the application:**
Open [http://localhost:3000](http://localhost:3000) in your browser.

### Key Features
- Server-Side Rendering (SSR) implementation for lightning-fast initial load times.
- Advanced JSON-LD Schema integration and optimized Next.js Image properties for high-scoring SEO.
- Pure CSS Grid construction with zero reliance on bulky CSS frameworks (no Tailwind or Bootstrap) to maintain a minimal DOM structure.
- Fully responsive architecture addressing specific layout shifts across Mobile (375px), Tablet (768px), and Ultra-wide Desktop (1440px) breakpoints.
